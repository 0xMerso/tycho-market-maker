Over 24 h with a $1 000 portfolio half in ETH:

If ETH rises 5 %: PnL_asset = 0.5×1000×0.05 = $25. If total value goes to $1 030 (ΔV = +30), arbitrage PnL = 30–25 = $5.
If ETH falls 5 %: PnL_asset = 0.5×1000×(–0.05) = –$25. If total value is $980 (ΔV = –20), arbitrage PnL = –20–(–25) = $5.

Over 72 h with a 10 % ETH move:

Up 10 %: PnL_asset = 0.5×1000×0.10 = $50; if ΔV = +70 then PnL_arb = 70–50 = $20.
Down 10 %: PnL_asset = –$50; if ΔV = –30 then PnL_arb = –30–(–50) = $20.

Over one week with a 20 % ETH swing:

Down 20 %: PnL_asset = –$100; if ΔV = –80 then PnL_arb = –80–(–100) = $20.
